# IDF Dashboard

Mobile-first Hebrew RTL dashboard (Next.js, TypeScript, Tailwind).

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS**
- No external UI libraries

## Structure

- `src/app/page.tsx` — main dashboard page
- `src/components/` — Header, SectionWrapper, Card, BenefitCard, ChartCircle, Footer, IconBar
